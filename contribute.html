    How to contribute to protoNN

    protoNN is an open-source project which you are very welcome to contribute to.

    If there's an issue you'd like us to fix, or question not covered in the tutorial, please raise an issue on Github.

    If you would like to collaborate and/or contribute some new functionality, get in touch! We like code that is pep8-compliant and comes with unit tests.
